# Slooze Front-End Challenge

Commodities Management System with role-based access control, authentication, and theme switching.

## Local Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Setup Instructions

1. Clone the repository:
```bash
git clone https://github.com/arjun248c/front-end-challenge.git
cd front-end-challenge
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
